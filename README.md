# Control Board Simulator

## 环境要求
- Node.js 18 或更新版本（附带 npm）

## 快速开始

先`cd app`，进入代码目录。

```bash
npm install       # 安装依赖
npm run dev       # 启动开发服务器（默认 http://localhost:5173）
```

开发模式才会显示 Manual Test 面板，所以日常调试直接执行 `npm run dev` 即可。

## 构建与预览
```bash
npm run build     # 生成生产包
npm run preview   # （可选）本地预览打包结果
```
